## [1.0.0] - 2025-01-01

### Added
- Initial release of VanityCert Widget
- Vanilla JavaScript implementation (no dependencies)
- 4-step SSL certificate provisioning flow
- LocalStorage state persistence per user
- Full theming support via CSS custom properties
- Backend proxy architecture for secure API integration
- Comprehensive callback system (6 events)
- Auto-polling for status updates
- 24-hour DNS validation timeout detection
- Domain format validation
- Copy-to-clipboard functionality
- Responsive design (mobile, tablet, desktop)
- Complete documentation and examples
- Backend proxy examples (Express.js, Laravel, Flask)
- Interactive demo page

### Features
- **Zero Dependencies**: Pure vanilla JavaScript
- **Customizable**: CSS variables for theming
- **Secure**: Backend proxy pattern (no API keys in browser)
- **Persistent**: LocalStorage state management
- **Responsive**: Mobile-friendly design
- **Accessible**: WCAG 2.1 compliant

### Components
- `vanitycert.js` - Main widget library
- `vanitycert.css` - Widget styles
- `vanitycert.min.js` - Minified JavaScript
- `vanitycert.min.css` - Minified CSS
